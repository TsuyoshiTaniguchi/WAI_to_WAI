# WAI to WAI

## サイト概要

### サイトテーマ
個人のスキル・経験・成長を記録・可視化し、自己PRやコミュニティ形成を支援するプラットフォーム  

### テーマを選んだ理由
自分のスキルや実績を効果的に発信する手段は限られており、従来の履歴書やポートフォリオでは十分に表現しきれない場合があります。  
特に、開発者やクリエイターにとって、GitHubの活動や技術スキル、プロジェクトの進捗を一元管理し、視覚的に魅力的に伝える場が必要だと考えました。  
この課題を解決するために、個人の成長を記録し、ストーリーとして伝えられるポートフォリオプラットフォーム「WAItoWAI」を開発しました。  

### ターゲットユーザ
- **開発者・クリエイター** → 自分のスキルや技術を発信し、キャリア形成したい人  
- **採用担当者・クライアント** → ユーザーのスキルやポートフォリオを簡単に確認したい人  

### 主な利用シーン
- **自分の技術・スキルを視覚的に発信したい時**  
- **ポートフォリオや実績を簡単に整理・記録したい時**  
- **同じ業界の仲間と繋がりたい時**
